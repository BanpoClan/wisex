# wisex
数据库管理系统。目标是实现所有的数据库表使用同一段代码进行增删改查，并且包含一对多的情况。同时，系统还能自动创建页面，此功能计划用Razor来实现。

以下代码结构，参考2015年在给HW做外包的框架。此代码框架在上面持续开发了7年，并没有做太大调整

web框架：采用MVC5

数据库连接：ibatis

IOC容器：StructureMap

消息队列：RabbitMQ

缓存：Redis
